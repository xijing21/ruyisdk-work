## Comments
### Files
Default：
```
/**
 * 
 */
```

Google Style：
```
/**
 * ${file_name}.
 */
```

### Types
Default：
```
/**
 * ${tags}
 */
```

Google Style：
```
/**
 * ${type_name}.
 *
 * ${tags}
 */
```
### Fields

Default：
```
/**
 * 
 */
```

Google Style：
```
*
 * The ${field_name}.
 */
```
### Constructors
Default：
```
/**
 * ${tags}
 */
```

Google Style：
```
/**
 * Constructs a new ${enclosing_type}.
 *
 * ${tags}
 */
```
### Methods
Default：
```
/**
 * ${tags}
 */
```

Google Style：
```
/**
 * ${enclosing_method}.
 *
 * ${tags}
 */
```

### Delegate methods
Default：
```
/**
 * ${tags}
 * ${see_to_target}
 */
```

Google Style：
```
/**
 * ${enclosing_method}.
 *
 * ${tags}
 * ${see_to_target}
 */
```
### Getters
Default：
```
/**
 * @return the ${bare_field_name}
 */
```

Google Style：
```
/**
 * Returns the ${bare_field_name}.
 *
 * @return the ${bare_field_name}
 */
```
### Setters
Default：
```
/**
 * @param ${param} the ${bare_field_name} to set
 */
```

Google Style：
```
/**
 * Sets the ${bare_field_name}.
 *
 * @param ${param} the ${bare_field_name} to set
 */
```
### Modules
Default：
```
/**
 * ${tags}
 */
```

Google Style：
```
/**
 * ${module_name}.
 *
 * ${tags}
 */
```
